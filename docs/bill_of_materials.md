# Liste des composants

## 📌 Composants communs aux deux versions
| Composant               | Quantité | Description                                                                 |
|-------------------------|----------|-----------------------------------------------------------------------------|
| Module RDA5807M         | 1        | Module FM I2C (76-108 MHz).                                                |
| Module Bluetooth MH-M18 | 1        | Module Bluetooth pour appairage smartphone.                                |
| Ampli PAM8403           | 1        | Amplificateur audio stéréo 3W.                                              |
| Haut-parleur 8Ω         | 1        | Haut-parleur pour sortie audio.                                            |
| Potentiomètre 10kΩ      | 2        | Potentiomètres linéaires pour le mode et la fréquence.                     |
| Axe Légo 3706           | 2        | Axes Technic pour les potentiomètres.                                       |
| Résistance 220Ω         | 1        | Pour la LED de rétroéclairage.                                             |
| LED 5mm                 | 1        | LED pour rétroéclairage.                                                    |
| Fils de connexion       | -        | Fils pour le câblage (Noir, Rouge, Orange, Vert, Bleu).                    |
